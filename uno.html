College projects

By Quim
Published on September 3, 2019

A list of older projects that I did as course projects or on my own. Note that they are either partially completed or not fully documented, as they were 100% experimental:

  • (Spanish) Danzabilidad: Charting the most successful dance hits in Spain on how much they can be used to dance. A explanation of the idea and how it is implented is written in GitHub.
  • (Spanish) Gender-based violence in Spain: An interactive graph displaying gender-based violence in Spain per region, between 2009 and 2016. A full-page version can be seen here.
  • (Catalan) Closest bike station in Barcelona: A course project I did at the Polytechnic School of Catalonia (UPC); written in Python 3. You can find the code and description here.
  • (English) Yet another concise document search engine: A course project I did at Charles University (CUNI) from Prague, Czech Republic; written in Python 3 and R. You can find the code and description here.
